# Expense Tracker 

A simple, lightweight web application built to log and track daily expenses. 

##  Features

* **Quick Logging:** Easily input the expense name and amount.
* **Real-Time Total:** Automatically calculates and displays the total spending.
* **Easy Management:** View logged items in a clean list and delete them with one click.

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
